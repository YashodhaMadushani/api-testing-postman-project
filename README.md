# API Testing Project using Postman

## Project Description
This project demonstrates API testing using ReqRes public API.

## Tools Used
- Postman

## Test Scenarios
- GET Users
- POST Create User
- PUT Update User
- DELETE User
- Negative Testing (404)

## Features
- Status code validation
- Response validation
- Automated test scripts
